# Projects

x-media (netflox)

Sharing media (videos only at current stage) platform, connected with TMDb and TVmaze. It gets file name and saves data from the third party APIs like poster, title, genres, rating etc. Based on the microservices architecture. Backend written in Golang, connected with MySQL and MongoDB. All containerized with Docker. Continuous Integration implemented with TravisCI.

Liobrus

Synergia Librus web client. One of my most time-consuming projects. First version had backend written in Python and Django. I used MySQL as database and Vue.js as frontend. I also used libraries like Chart.js or AOS.js. Version I'm currently working on will have Golang backend, MongoDB database. Frontend will be written in React, also I'll use microservices architecture.

DDS ($$)

DDS (double dollar sign) is a web platform, which helps companies and individuals manage their expenses, incomes, generating and sending invoices, etc. Backend written in Golang, connected with MongoDB. Frontend and invoices theme generator written in React. Containerized with Docker. Continuous Integration implemented with CircleCI.

GoLibrusAPI

Simple and fast Synergia Librus API wrapper written in Golang.

GO_FS

Easy to use HTTP file server written in Golang and containerized with Docker.
